Projects

Detecting and Localizing Tumors in Brain MRI Scans (Capstone Project)

Utilizing machine learning algorithms to accurately detect and localize brain tumors in MRI scans, this capstone project addresses the critical need for early diagnosis and treatment. The proposed solution involves training convolutional neural networks and deep residual U-nets on MRI data to achieve statistically significant performance in tumor detection and segmentation. Through rigorous evaluation, including metrics such as accuracy, precision, and the Dice similarity coefficient, the project demonstrates the potential of machine learning in revolutionizing brain tumor diagnostics.

Statistical Evaluation of Website Performance through A/B Testing

This project involves conducting an A/B test to determine if an e-commerce business should launch a new website. A database is created to track conversions from both the old and new sites, calculating the probability of conversions and generating 10,000 outcomes using numpy's random choice. These outcomes are tested for conversion rates, and the analysis includes finding the p-value and z-score, as well as applying several regression models with additional terms. The results of this thorough testing will reveal whether the new page offers any significant improvement over the existing site.

Link to GitHub

Comprehensive Analysis of Bike Sharing Usage Patterns

This project involves analyzing the usage patterns of the Ford Go Bike system in the greater San Francisco Bay area. By utilizing Python and various data analysis libraries, the project aims to uncover insights into customer habits. Key tasks includes calculating the distance between coordinates using the haversine formula and performing univariate, bivariate, and multivariate explorations. Visualizations are created to highlight key findings, such as the travel habits of different demographic groups.

Analysis of Content Engagement on Social Media Platforms

This project demonstrates proficiency in gathering, cleaning, and analyzing data from the Twitter account "We Rate Dogs". The analysis focuses on identifying insights such as which dog breeds or ages received the highest engagement on posts. Visualizations are used to highlight trends and patterns in user interactions with dog-related content.
